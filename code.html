Programming Vocabulary Guide

Essential vocabulary to start programming in R, SQL, Python, and C.

R

  • Data Frame: A table or 2D array in R, where columns can have different types.
  • Vector: A sequence of data elements of the same basic type.
  • Function: A block of code in R designed to perform a specific task.
  • Library: A collection of functions and datasets for specific tasks.

SQL

  • Query: A request for data or information from a database.
  • Table: A structured set of data held in rows and columns.
  • JOIN: A SQL operation to combine data from two or more tables.
  • Primary Key: A unique identifier for each row in a SQL table.

Python

  • Variable: A name that holds a value.
  • Loop: A structure that repeats a set of instructions until a condition is met.
  • Function: A reusable block of code designed to perform a specific task.
  • Library: A collection of pre-coded functions and methods.

C

  • Pointer: A variable that stores the memory address of another variable.
  • Array: A collection of items stored at contiguous memory locations.
  • Function: A block of code that performs a particular task.
  • Header File: A file that provides definitions and declarations to the code.
